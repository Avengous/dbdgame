# dbdgame
Host: https://dbdgame.herokuapp.com/

Dependencies:
- https://nodejs.org/en/
- https://phaser.io/phaser3
- https://socket.io/

Tools:
- Tiled Map Editor https://www.mapeditor.org/

Setup Instructions:
- Install NPM
- Clone repository
- CD to repository
- Run "npm install"
- Start server with "npm start"
